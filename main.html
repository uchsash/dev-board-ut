Question-1: What are the different ways to select an element in the DOM?
There are several different ways i.e. getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() and querySelectorAll(). Here, getElementById() and querySelector() select single element and all the rest is used to select multiple elements.
Question-2: What is the difference between innerHTML, innerText, and textContent?
innerHTML gives all the HTML content inside of an element, while innerText gives only the visible text, and textContent returns all the text without considering visiblity and formatting.
Question-3: What is event delegation in the DOM?
event delegation is adding an event listener (addEvenetListener()) to the parent which will handle all the elements inside its child, instead of adding different event listener for each child element.
Question-4: What is event bubbling in the DOM?
Event bubbling is a type of event propegation in the DOM, which means it holds the target child and then go upwards like creating a bubble and reach to the parent. That way event handler can work from the root or parent element and handle the event necessary for the child element.
Question-5: How do you create, add, and remove elements using JavaScript?
In JavaScript, to create an element, we use createElement() method. To add an element, we use append() or appendChild() method. And to remove an element, we use remove() or removeChild() method.
